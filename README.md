# 🌍 EcoTrack AI (Neo-Terra)

> "Optimizing the human-biosphere matrix, one variable at a time."

EcoTrack AI is a disruptive, retro-futuristic sustainability platform designed to calculate, monitor, and algorithmically optimize human carbon outputs. Engineered with a brutalist "glitch-art" aesthetic and a high-performance React frontend, EcoTrack gamifies carbon neutralization to encourage rapid behavioral shifts in the user matrix.

## ⚡ Core Features

- **Algorithmic Telemetry Ingestion (Tracker)**: Securely log daily variables across Transport, Dietary, Energy, and Waste sectors.
- **Global Status Hub**: Real-time visualization of your Eco Score and Gross CO2e output.
- **AI Directives Coach**: A synthetic neural network providing specific, actionable protocols to lower footprint and maximize economic retention.
- **Gamified Progression Engine**: Rank up your Sustainability Level, maintain daily streaks, and unlock achievements.
- **Historical Data Matrix**: Deep-dive analytics with cross-sectional breakdown charts of carbon loads.

- **Frontend Core**: React 18, Vite, React Router, Tailwind CSS, Recharts
- **Backend Core**: Python, FastAPI, SQLAlchemy, SQLite
- **Testing**: Vitest, React Testing Library, Pytest

## 🏆 Innovation & Impact

EcoTrack AI represents a paradigm shift in climate-action UX. Moving away from standard sterile corporate designs, EcoTrack employs a striking, high-contrast, cyberpunk UI that transforms carbon reduction from a chore into a high-stakes tactical mission, increasing user engagement and long-term retention.
